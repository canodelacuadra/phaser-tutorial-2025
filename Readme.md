# Tutorial phaser
Tutorial oficial de phaser  que es un juego en el que el jugador corre y salta entre plataformas, recoge estrellas y evita a los enemigos. Durante este proceso, explicaremos algunas de las características principales del framework.

La única diferencia es que está lanzado con vite

Para instalar clonar o  descargar
````shell
npm init
npm run dev

 ````
 y lo podemos ver en ejecución en
 
  http://localhost:5173
