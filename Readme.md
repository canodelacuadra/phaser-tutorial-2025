# Tutorial phaser
Tutorial oficial de phaser  que es un juego en el que el jugador corre y salta entre plataformas, recoge estrellas y evita a los enemigos. Durante este proceso, explicaremos algunas de las características principales del framework.

La única diferencia es que está lanzado con vite y hecho el build en Netlify
[Ver en Netlify ](https://phaser-tutorial-canodelacuadra.netlify.app/)

Para instalar clonar o  descargar
````shell
npm init
npm run dev

 ````
 y luego  lo podemos ver en ejecución en el servidor de desarrollo de vite en nuestro equipo
 
  http://localhost:5173
